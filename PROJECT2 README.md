# Customer Churn & Retention Analysis Dashboard

## Overview

This Power BI dashboard analyzes customer churn behavior, retention performance, and customer engagement patterns. The goal is to identify churn drivers and recommend strategies to improve customer retention.

## Objectives

- Measure customer churn rate
- Analyze churn trends over time
- Identify major churn reasons
- Compare churn across plans and industries
- Evaluate cohort retention performance

## Key Metrics

- Total Customers: 500
- Active Customers: 390
- Churned Customers: 110
- Customer Churn Rate: 22%
- Total Subscriptions: 5K

## Dashboard Features

- KPI Cards
- Churn Trend Analysis
- Churn Reason Analysis
- Plan-wise Churn Analysis
- Industry-wise Churn Analysis
- Referral Source Analysis
- Cohort Retention Analysis

## Key Insights

- Customer churn rate reached 22%.
- Missing features were the leading churn reason.
- Budget limitations and support issues contributed significantly to churn.
- Pro and Basic plans recorded the highest churn.
- Retention performance varied across customer cohorts.

## Recommendations

- Improve product features based on customer feedback.
- Strengthen customer support services.
- Implement retention campaigns for high-risk customers.
- Introduce pricing and loyalty strategies.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
